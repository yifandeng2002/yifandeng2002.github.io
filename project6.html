Overview

This project focuses on developing a physical device and a real-time, web-based dashboard to monitor and visualize environmental sound levels using a microcontroller, microphone (MAX9814), and network communication via MQTT. The system captures both sound intensity and frequency data, displaying insights through an interactive UI built with HTML, CSS, and JavaScript (Chart.js).
