Overview

SoniBox is an embedded tangible user interface designed to control generative soundscapes by mapping user touch on a tactile interface to emotional dimensionsâ€”calmness and vibrancy. The interface consists of a textile-based x-y grid made from conductive material, with ESP microcontrollers for touch sensing and a Raspberry Pi running Sonic Pi for real-time audio synthesis. The system communicates via Open Sound Control (OSC). This research focuses on HCI and computational music and how they integrate to create interactive IoT system.
